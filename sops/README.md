# SOP Library
Version: 1.0
Last Updated: 2025-05-06
Purpose: Centralize Oracle Boxing standard operating procedures and templates.

---
## How to Use
- Create new SOPs per department using the templates in `templates/`.
- Include version, owner, and last updated date on every SOP.
- Link SOPs to related KPIs and projects where applicable.

## Contents
- `templates/` – SOP and process flow templates.
- `marketing/` – Marketing SOPs.
- `operations/` – Operational SOPs.
- `product/` – Product & coaching SOPs.
