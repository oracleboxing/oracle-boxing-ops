# KPI Library
Version: 1.0
Last Updated: 2025-05-06
Purpose: Maintain Oracle Boxing company and departmental KPIs with definitions and owners.

---
## How to Use
- Update KPI values on the agreed cadence (weekly/monthly).
- Add or retire KPIs using `kpi_template.md`.
- Link KPIs to responsible roles and related projects.

## Contents
- `company_kpis.md` – Top-level metrics.
- `departmental_kpis.md` – Metrics by department.
- `kpi_template.md` – Add new KPIs with consistent formatting.
