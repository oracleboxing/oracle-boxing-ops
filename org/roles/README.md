# Roles Overview
Version: 1.0
Last Updated: 2025-05-06
Purpose: Central place for role definitions, scopes, and expectations for Oracle Boxing.

---
## How to Use
- Create or update a role file when scope or reporting lines change.
- Keep responsibilities aligned with departmental objectives and KPIs.
- Include decision rights, success metrics, and key collaboration points.

## Template
- Overview of the role
- Key responsibilities (list)
- Decision rights
- KPIs / success measures
- Notes and open questions
