## Scope

Own business reliability: customer support, payment processing, automations, system uptime, tax compliance, and contractor management. Ensure the operational foundation enables all other departments to function smoothly.

Jordan currently holds this role alongside CEO and Growth Operator. Mid-term goal is to delegate to a dedicated hire.

---

## Key Responsibilities

### Customer Support
- [ ] Answer customer emails daily
- [ ] Process refunds promptly when appropriate
- [ ] Resolve access issues and login problems
- [ ] Handle member complaints and escalations

### Financial Management
- [ ] Track profit and loss monthly
- [ ] Pay contractor wages and commissions monthly
- [ ] Maintain budget tracking and cash flow visibility
- [ ] Process expense tracking and reporting

### Tax & Compliance
- [ ] File quarterly VAT return
- [ ] File annual corporation tax return
- [ ] File annual self assessments
- [ ] File monthly PAYE for directors

### Systems & Automation
- [ ] Ensure all Make.com automations are functional (weekly review)
- [ ] Monitor Stripe payment flows and webhook health
- [ ] Review automation error logs and fix issues
- [ ] Ensure website is operating functionally (weekly checks)
- [ ] Verify membership portal access is working

### Incident Management
- [ ] Maintain incident log for system issues
- [ ] Document runbooks for common issues
- [ ] Troubleshoot software errors and bugs promptly
- [ ] Escalate technical issues to Dev/Tech when needed

### Contractor Management
- [ ] Engage Dev/Tech contractor via SOW for projects
- [ ] Maintain backlog of integrations and technical improvements
- [ ] Review and accept completed technical work
- [ ] Maintain change log for automations and integrations

### Business Improvement
- [ ] Identify inefficiencies that can be removed or automated
- [ ] Propose and implement process improvements
- [ ] Maintain operational documentation

---

## Decision Rights

### Decides Without Asking
- Support response approach and tone
- Refund decisions within policy
- Automation fixes and minor changes
- Contractor task prioritisation (within budget)
- Routine operational decisions

### Must Escalate
- Refunds outside policy (CEO)
- Significant system outages (CEO + affected departments)
- Major automation changes (CEO)
- Contractor budget increases (CEO)
- Compliance concerns (CEO / legal)

---

## KPIs / Success Measures

- **Support response:** Inbox cleared daily
- **System uptime:** Sites and automations functional
- **Payment processing:** No failed payments due to system issues
- **Tax compliance:** All filings on time
- **Contractor payments:** Paid on time monthly
- **Incident resolution:** Issues resolved within SLA
