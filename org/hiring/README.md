# Hiring Folder Guide
Version: 1.0
Last Updated: 2025-05-06
Purpose: Store hiring roadmap, job scorecards, and recruitment documentation.

---
## How to Use
- Keep the hiring roadmap current by quarter.
- Create a scorecard for every open role using the template.
- Link candidates and interview notes to the relevant scorecard.

## Contents
- `hiring_roadmap.md` – Planned roles and timing.
- `job_scorecard_template.md` – Template for defining success for new hires.
