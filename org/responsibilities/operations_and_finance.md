# Operations & Finance Responsibilities

**Version:** 2.0
**Last Updated:** 2025-12-13
**Purpose:** Track the core responsibilities of the Operations & Finance department.

---

## Department Overview

Ensures business reliability through customer support, payment processing, automations, system uptime, tax compliance, and contractor management.

---

## Responsibility List

| Responsibility | Owner | Cadence | Status/Metric |
| --- | --- | --- | --- |
| Answer customer emails | Jordan | Daily | Inbox cleared daily |
| Process refunds where appropriate | Jordan | As needed | Refunds processed promptly |
| Resolve access issues and login problems | Jordan | As needed | Issues resolved < 24h |
| Track profit and loss | Jordan | Monthly | P&L report completed |
| Pay contractor wages and commissions | Jordan | Monthly | Payments on time |
| Ensure Make.com automations are functional | Jordan | Weekly | Automations running |
| Monitor Stripe payment flows | Jordan | Weekly | Payments operational |
| Ensure website is operating functionally | Jordan | Weekly | Site online |
| Verify membership portal access | Jordan | Weekly | Portal accessible |
| Troubleshoot software errors and bugs | Jordan | As needed | Issues resolved |
| File quarterly VAT return | Jordan | Quarterly | Filed on time |
| File annual corporation tax return | Jordan | Annually | Filed on time |
| File annual self assessments | Jordan | Annually | Filed on time |
| File monthly PAYE for directors | Jordan | Monthly | Filed on time |
| Identify inefficiencies to remove or automate | Jordan | Ongoing | Improvements made |
| Engage Dev/Tech contractor via SOW | Jordan | As needed | SOW agreed |
| Maintain incident log and runbooks | Jordan | Ongoing | Documentation current |

---

## Notes

- Maintain support/resolution log and automation change log to track reliability
- Dev/Tech contractor engaged on project basis; see role file for engagement model
- Tax schedule: PAYE monthly, VAT quarterly, Corp tax annually, SA annually
