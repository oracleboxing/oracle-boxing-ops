# Oracle Boxing – CEO Operating System
Version: 1.0
Last Updated: 2025-05-06
Purpose: Central repository for Oracle Boxing organisational structure, operating system, and execution frameworks.

---
## What Lives Here
- Org design, roles, responsibilities, deliverables
- SOP library with templates
- KPIs and measurement cadences
- Project and planning artifacts
- Finance runbooks and checklists

## How to Use
- Start with `org/organisational_blueprint.md` to understand the company model.
- Update role and responsibility files whenever team membership changes.
- Keep SOPs current; version and date every update.
- Track KPIs weekly/monthly; use templates to add new metrics.
- Log major initiatives under `projects/` and archive outcomes.

## Update Cadence
- Weekly: KPI updates; project status.
- Monthly: Finance checklists; SOP revisions as needed.
- Quarterly: Org blueprint review; hiring roadmap refresh; project planning.

## Folder Guide
- `org/` – Structure, roles, responsibilities, deliverables, hiring.
- `sops/` – Standard operating procedures and templates.
- `kpis/` – Company and departmental metrics.
- `projects/` – Planning templates and initiative trackers.
- `finance/` – Accounting, payroll, VAT, and cashflow processes.
